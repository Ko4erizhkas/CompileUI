# CompileUI
## Лабораторные работы по дисциплине "Теория формальных языков и компиляторов"
## Лабораторная работа 1 
1. Цель работы - создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора
2. Данную работу выполнил студент Косяченко Даниил из группы АВТ-313
3. В приложении реализованы необходимые кнопки и поля.
4. Использовался язык программирования - С++ + QML (Qt)
5. Для запуска программы необходимо иметь:
  5.1.1. Visual Studio 2022\2026
  5.1.2. CMake
  5.1.3. Статическую сборку Qt 6.10.2
  5.1.4. Компиляторо с поддержкой C++17
  5.1.5. x64 Native Tools Command Prompt for VS 2022
  5.2 - Сборка проекта
  5.2.1 - ```bash cmake .. -G Ninja -DC_MAKE_BUILD_TYPE=Release "-DQt6_DIR=ДИРЕКТОРИЯ_С_СТАТИЧЕСКОЙ_СБОРКОЙ_Qt_\lib\cmake\Qt6"```
  5.2.2 - ```bash cmake --build . --parallel```
6. Руководство пользователя - отсутствует
7. Интерфейс имеет чисто визуальный характер без необходимого функционала

Интегрировать разработанный модуль в ранее созданный графический интерфейс языкового процессора
2. Реализован собственный сканер для прототипов функций С++
