Меню "Файл"
Создать - открывает новый пустой файл
Открыть - открывает имеющийся на компьютере файл в формате .txt
Сохранить - если редактируемый файл имеет место сохранения на компьютере, то редактированный файл сохраняется со всеми изменениями вместо старого. Если файл сохраняется впервые, то открывается диалоговое окно, предлагающее пользователю сохранить файл в любом месте в формате .txt
Сохранить как - открывает диалоговое окно, предлагающее пользователю сохранить редактируемый файл в удобном для него месте
Закрыть вкладку - закрывает текущую вкладку
Выход - выход из приложения

Меню "Правка"
Отменить - отменяется последнее действие пользователя - Ctrl+Z
Вернуть - восстановление ранее отмененного действия пользователя - Ctrl+Y
Вырезать - сохранение выделенного текста в буфере обмена с последующим его удалением - Ctrl+X
Копировать - сохранение выделенного текста в буфере обмена - Ctrl+C
Вставить - занесение в поле ввода сохраненного в буфере обмена текста - Ctrl+V
Удалить - удаление выделенного текста - Del
Выделить все - выделяет весь текст в поле ввода - Ctrl+A

Меню "Текст"
Постановка задачи - открывает .txt файл и выводит текст с постановкой задачи в поле общего вывода программы
Грамматика - открывает .txt файл и выводит текст с грамматикой в поле общего вывода программы
Классификация грамматики - открывает .txt файл и выводит текст с классификацие Хомского в поле общего вывода программы
Метод анализа - открывает .txt файл и выводит текст с методом анализа в поле общего вывода программы
Тестовый пример - открывает .txt файл и выводит текст с тестовыми примерами в поле общего вывода программы
Список литературы - открывает .txt файл и выводит текст со списком литературы в поле общего вывода программы
Исходный код программы - открывает .txt файл и выводит ссылку на публичный репозиторий с исходным кодом программы в поле общего вывода программы

Меню "Пуск"
Запуск лексера - запускает процесс лексического анализатора введённого кода. Результат выводится в окне "Таблица токенов"
Запуск парсера - запускает процесс синтаксического анализатора введённого кода. Резальтат выводится в окне "Ошибки парсера" в случае нахождения ошибок. Иначе выводит в общем поле вывода - "Ошибок нет" 

Меню "Справка"
Вызов справки - описание реализованных функций (текущий документ)
О программе - описание приложения
