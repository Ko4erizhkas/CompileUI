Прототип функции в языке C/C++ – это объявление функции, задающее тип возвращаемого значения, имя функции и список параметров без реализации тела. Прототип завершается символом «;».

Тип возвращаемого значения и типы параметров задаются ключевыми словами. В рамках курсовой работы поддерживаются следующие типы: void, int, short, float, double. Имя функции и имена параметров являются идентификаторами — последовательностями латинских букв, цифр и символа подчёркивания («_»), начинающимися с буквы. Каждый параметр в списке задаётся парой «тип идентификатор» и отделяется от следующего запятой.

Для объявления прототипа функции используется один из ключевых слов типа возвращаемого значения: void, int, short, float, double.

Формат записи: "тип идентификатор(тип идентификатор, ..., тип идентификатор);".

В связи с разработанной автоматной грамматикой G[Z] синтаксический анализатор (парсер) будет считать верными следующие конструкции:
1. int dada(int nono);
2. short nono(int a, short b, float c);
3. void net();
